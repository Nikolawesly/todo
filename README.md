# TO-DO APP

A full-stack To-Do Application developed using MERN Stack technologies.  
The application allows users to register, login, create tasks, update tasks, and manage daily activities efficiently.

---

## Features

- User Registration & Login Authentication
- Create, Update & Delete Tasks
- Task Management Dashboard
- JWT Token Authentication
- Responsive User Interface
- REST API Integration
- MongoDB Database Connectivity

---

## Technologies Used

### Frontend
- React.js
- Vite
- CSS

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication
